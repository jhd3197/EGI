# Code of Conduct

This project is for people helping people. We expect everyone to act with empathy, respect, and patience.

## Our standards

- Treat others with kindness.
- Welcome newcomers and answer questions.
- Accept constructive criticism gracefully.
- Focus on what helps the people affected by disasters.
- Do not harass, discriminate, or intimidate anyone.
- Do not use the project to spread misinformation or exploit vulnerable people.

## Enforcement

If someone violates this code of conduct, maintainers may remove comments, close issues, or ban contributors. Report problems by emailing or opening a private issue if available.

## Scope

This code applies to all project spaces: GitHub, chat channels, email, events, and anywhere the project is discussed.
